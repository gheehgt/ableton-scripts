quick and dirty ableton live script for the dj techtools midifighter twister

----------------------------------------------------------------------------

1. load up the 'gtroll_twister_public.mfs' file via the twister utility app and dumb to your twister for the right config settings

2. place the â€˜gtroll_twister' folder into ableton lives midi remotescripts

3. fire up your set

----------------------------------------------------------------------------
-------
CONTENT
-------

bank 1: device control(blue hand), device nav, track select, scene up/down, scene start, play, stop, overdub, 8x volume;
bank 2: mixer assignments for track 1-4
bank 3: mixer assignments for track 5-8
bank 4: ff proG assignments for track 1,2,7,8

sidebuttons: bank select + sequencer mode

note1: to be used with ableton default set 'plasma'
